=== Example 1: Simple Binary Budget Aggregation ===
Input:
  budget = 100
  valuation_matrix = [[100, 0, 0], [0, 0, 100]]
  
Explanation:
  - Agent 1 values: Project 0 (100), Project 1 (0), Project 2 (0)
  - Agent 2 values: Project 0 (0), Project 1 (0), Project 2 (100)
  - Total budget: 100
  
Expected Output:
  compute_budget(100, [[100, 0, 0], [0, 0, 100]])
  
Analysis:
  - Agent 1 only cares about Project 0 with full value 100
  - Agent 2 only cares about Project 2 with full value 100
  - Since budget = 100, we can only fund one project completely
  - Optimal allocation should balance utilities or maximize social welfare
  - Possible solutions: 
    * Fund Project 0 completely (satisfies Agent 1)
    * Fund Project 2 completely (satisfies Agent 2)
    * Split budget between projects based on fairness criteria
